School Management System ==>

The School Management System is a comprehensive web-based application developed using Python Django and MySQL.

Installation ==>
 Navigate to the project directory: cd school-management-system
 create virtual envronment -->python -m venv project
create a project --> cd project/Scripts 

->Install the project dependencies

-> Set up the database: 

Install MySQL and create a new database.

Update the database configuration in the settings.py file.



--> Run database migrations:

--> Create a superuser (admin account)

--> Start the development server: python manage.py runserver
